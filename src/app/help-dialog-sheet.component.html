Keyboard Interaction

Zoom and Pan

ALT + WHEEL - Time Pan
CTRL + WHEEL - Amplitude Zoom
SHIFT + WHEEL - Time Zoom

Mode

z - Toggle Common Time Mode
x - Toggle Common Amplitude Mode
p - Toggle Predicted Picks

Picking

d - Toggle P Picking Mode
f - Toggle S Picking Mode
LEFT in Picking Mode / CTRL+LEFT on Pick / CONTEXT - Remove Pick
MID in P Picking Mode / CTRL+MID - Add / Move P Pick
MID in S Picking Mode / SHIFT+MID - Add / Move S Pick
