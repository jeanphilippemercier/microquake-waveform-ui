Keyboard Interaction

Zoom and Pan

ALT + WHEEL - Time Pan
CTRL + WHEEL - Amplitude Zoom
SHIFT + WHEEL - Time Zoom

Mode

z - Toggle Common Time Mode
x - Toggle Common Amplitude Mode
p - Toggle Predicted Picks

Picking

d - Toggle P Picking Mode
LEFT in P Mode / CTRL+LEFT / MENU - Create / Move P Pick
MID in P Mode / CTRL+MID / MENU - Remove P Pick
f - Toggle S Picking Mode
LEFT in S Mode / SHIFT+LEFT / MENU - Create / Move S Pick
MID in S Mode / SHIFT+MID / MENU - Remove S Pick
s - Undo Last Picking Operation
