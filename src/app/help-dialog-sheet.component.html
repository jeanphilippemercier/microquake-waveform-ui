Interaction

Zoom and Pan

ALT + WHEEL - Time Pan
CTRL + WHEEL - Amplitude Zoom
SHIFT + WHEEL - Time Zoom

Mode

z - Toggle Common Time Mode
x - Toggle Common Amplitude Mode
p - Toggle Predicted Picks

Picks

CTRL+LEFT on Pick / CONTEXT MENU - Remove Picks
CTRL+MIDDLE - Add P Pick at current position
SHIFT+MIDDLE - Add S Pick at current position
