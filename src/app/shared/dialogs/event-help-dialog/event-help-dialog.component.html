Help

Zoom and Pan
ALT + Mouse Scroll - Time Pan
CTRL + Mouse Scroll - Amplitude Zoom
SHIFT + Mouse Scroll - Time Zoom
Picking
d - Toggle P Pick Mode
f - Toggle S Pick Mode
s - Undo Last Picking Operation
LEFT in P Mode / CTRL + LEFT / Context - Create / Move P Pick
RIGHT in P Mode / CTRL + RIGHT / Context - Remove P Pick
LEFT in S Mode / SHIFT + LEFT / Context - Create / Move S Pick
RIGHT in S Mode / SHIFT + RIGHT / Context - Remove S Pick
Arrow R/L / Mouse Scroll Down/Up in P/S Mode - Move Pick to Right/Left
SHIFT + Right/Left Arrow in P/S Mode - Move Pick Fast to Right/Left
Navigation
a - Previous Page
s - Next Page
Evaluation
q - Set Event as Rejected
w - Set Event as Confirmed
e - Toggle Event Edit Dialog
z - Open / Close Labelling Mode
x - Select / Deselect All Traces (in Labelling Mode)
1 - 9 - Toggle Label (in Labelling Mode)
Other
v - Toggle Common Time Mode
b - Toggle Common Amplitude Mode
c - Toggle Display Entire Traces
p - Toggle Predicted Picks
r - Toggle Accepted/Rejected Event Status
h - Toggle Help
